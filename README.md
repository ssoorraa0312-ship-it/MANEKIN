# Aether Apparel (MANEKIN) - Polished Brutalism LP

A high-fidelity fashion e-commerce experience built with Next.js 15, embodying the "Polished Brutalism" aesthetic.

## Features

- **Immersive Hero**: Kinetic typography ("MANEKIN") with glitch effects and decoder subtitles.
- **The Archive**: A catalogue of "Invisible Man" fashion items displayed in a high-density, museum-like grid.
- **System Aesthetic**: A "Purified" interface with a system status bar, removing traditional navigation paradigms.
- **Micro-Interactions**: Custom magnetic cursor, noise overlay, and scroll-driven parallax effects.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS + CSS Modules
- **Animation**: Framer Motion
- **Fonts**: Syne, Playfair Display, Inter

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploy on Vercel

The easiest way to deploy this Next.js app is to use the [Vercel Platform](https://vercel.com/new).

1. Push this code to a GitHub repository.
2. Connect the repository to Vercel.
3. It will deploy automatically with zero configuration.
